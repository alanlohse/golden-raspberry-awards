# Golden Raspberry Awards API

### Building

Run the following command on the prompt:

    ./mvnw clean package

Building will automatically run the tests.

### Running tests

The integration tests are implemented by a junit test case. It can be run using the command below:

    ./mvnw test

### Calling the API

The API can be called through the http address [http://localhost:8080/api](http://localhost:8080/api)
